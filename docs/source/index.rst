Build instructions
##################

RanDoc is available on https://github.com/grembeter/randoc. You need :program:`python3` to built it
from scratch:

.. code-block:: shell

   git clone https://github.com/grembeter/randoc
   cd randoc
   source ./build-env

Open :file:`docs/build/html/index.html` page in browser.
